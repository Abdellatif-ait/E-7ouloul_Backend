# E-7ouloul_Backend
This is the backend of an Acadimic project the we worked on during this year.It aims to make tourism in algeries easier.
# How to build the Project
```
  npm i
```
```
  npm run start 
```
```
  npm run dev
```
# How to use it
   ## Routes
   ```
     '/tourist'
      '/' GET
      '/:id' GET
      '/login' POST
      'register' POST 
      '/follow' POST,DELETE
   ```
